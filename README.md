# InsuranceDemo

Insurance consultancy marketing site (Next.js): **Marula Insurance Advisory** — hero, services, quote form, WhatsApp CTA, contact map.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Config

- Brand, contact, WhatsApp, Unsplash image URLs: `src/lib/site-data.ts`
- Remote images: `images.unsplash.com` is allowed in `next.config.ts`

## Stack

Next.js 16, React 19, Tailwind CSS 4.
